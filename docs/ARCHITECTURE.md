## Flusso schermata per schermata

1. **HomeScreen**
   - Headline emozionale
   - CTA "Inizia"
2. **EventTypeScreen**
   - Scelta immediata: Sala EVE / Location esterna
   - Card tappabili, micro animazione
3. **LocationScreen** (se Location esterna)
   - Inserisci indirizzo, data, numero bambini
   - Motore suggerimenti: badge, consigli personalizzati
4. **SalaScreen** (se Sala EVE)
   - Scelta sala: Luxuri (dominante), Premium, Family
   - Valore reale calcolato dinamicamente
5. **ExtraScreen**
   - Sistema extra stile Glovo: categorie, icone, descrizione, prezzo, +/-, animazione bounce, sticky bar

## Design System
- Palette vibrante, accenti soft, ombre, angoli arrotondati
- Tipografia bold, microcopy leggera, gerarchia chiara

## Motion System
- Stagger entrance, slide-up, animated price counter, sticky bar animata, feedback bounce

## Strategie Upsell Etiche
- Bundle suggeriti, upgrade evidenziati, extra coerenti, valorizzazione intelligente

## Logica suggerimenti dinamici
- Suggerimenti in base a numero bambini, tipo evento, preferenze
- Micro rinforzi positivi: "Ottima scelta", "Evento premium in costruzione"
